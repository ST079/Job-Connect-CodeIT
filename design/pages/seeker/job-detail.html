Job Description

We're looking for a passionate Senior Frontend Developer to join our product team at TechCorp. You'll be responsible for building the next generation of our web applications, working closely with designers and backend engineers to create exceptional user experiences.

Responsibilities

  • Develop and maintain responsive web applications using React.js
  • Collaborate with UI/UX designers to implement pixel-perfect designs
  • Write clean, maintainable, and efficient code following best practices
  • Optimize applications for maximum speed and scalability
  • Mentor junior developers and conduct code reviews
  • Stay up-to-date with emerging frontend technologies and trends

Technologies You'll Work With

React.js TypeScript Next.js Tailwind CSS GraphQL Jest/Testing Library Webpack Git/GitHub

What You'll Build

You'll be working on our flagship product used by over 50,000 businesses worldwide. The platform helps companies streamline their operations through intelligent automation and real-time analytics.

Requirements

Must Have

  • 3+ years of professional experience with React.js and modern JavaScript
  • Strong proficiency in TypeScript and modern ES6+ features
  • Experience with state management libraries (Redux, Zustand, or Context API)
  • Solid understanding of responsive design and cross-browser compatibility
  • Experience with testing frameworks (Jest, React Testing Library, Cypress)

Nice to Have

  • Experience with Next.js or other SSR frameworks
  • Knowledge of GraphQL and Apollo Client
  • Familiarity with CI/CD pipelines and DevOps practices
  • Contributions to open-source projects

Soft Skills

  • Excellent communication and collaboration skills
  • Problem-solving mindset and attention to detail
  • Ability to work in a fast-paced agile environment
